Shared memory in JavaScript: Sundry demos

These are some older demo programs coming out of the shared memory work. Some of these are dated and ad-hoc and not all that interesting, but they should all work (last tested on 2016-10-19 in Firefox 50b7 on MacOS X 10.11). The distinctions among some of the programs will be lost on you unless you study the source code and the programs' README.md files. For most of these, you can vary the number of workers by adding the ?workers=n parameter in the URL; the default is 4.

More current demos and benchmarks are in the repo https://github.com/lars-t-hansen/parlib-simple, in the demo/ directory.

NOTE ABOUT GOOGLE CHROME: Google Chrome should be able to run these but will not allow programs loaded from file: URLs to create web workers. You must therefore host them on a local web server. See README-CHROME.md for more.
